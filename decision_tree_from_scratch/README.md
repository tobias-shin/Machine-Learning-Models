# Decision Tree From Scratch

This is a small, dependency-free decision tree classifier built from scratch in Python.

## Files

- `decision_tree.py`: the classifier implementation
- `example.py`: a runnable demo
- `test_decision_tree.py`: a lightweight test script

## Run

```bash
cd /Users/tobiasshin/Desktop/anaconda_projects/decision_tree_from_scratch
python3 example.py
python3 test_decision_tree.py
```

## What it supports

- binary recursive splits on numeric features
- Gini impurity
- information gain
- `fit`, `predict`, `score`, and `print_tree`
